# RailMind Architecture

RailMind is organized as a small workspace monorepo for a 3-day hackathon MVP.

## Folders

- `frontend/`: Next.js 14 dashboard with Tailwind CSS.
- `backend/`: Express API and Socket.IO server.
- `shared/`: Shared TypeScript types and constants.
- `docs/`: Product and engineering notes.

## Runtime Shape

- The frontend runs on `localhost:3000`.
- The backend runs on `localhost:4000`.
- Socket.IO is hosted by the backend.
- Gemini and Neon integrations are represented by placeholder clients.
